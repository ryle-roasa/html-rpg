# html-rpg
A beginner RPG game using mostly JavaScript and HTML canvas.
---------
Controls
---------
Movement:
"W" to move up
"A" to move left
"S" to move down
"D" to move right

Attack:
"Left click" to attack (basic attack)
"Right click" to attack (special attack)

Aiming:
Use mouse to aim.
